# Retail Operations Manager — Store Performance Turnaround

## Project Overview

This project simulates a real-world retail operations turnaround initiative focused on improving store performance, reducing operational inefficiencies, increasing team accountability, and improving KPI visibility for store leadership.

The project was built from the perspective of a Retail Operations Manager responsible for identifying performance gaps, aligning stakeholders, tracking store-level KPIs, reducing execution issues, and creating a repeatable improvement plan.

## Business Problem

The store was experiencing inconsistent sales performance, missed operational tasks, low team accountability, and limited visibility into daily performance metrics. Leadership needed a structured turnaround plan that connected store execution, staffing, customer experience, and KPI tracking.

## Project Goals

- Improve store sales performance through better daily execution
- Increase team accountability using clear task ownership
- Track key retail KPIs across sales, conversion, labor, and customer experience
- Reduce missed operational tasks and follow-up delays
- Create a manager-ready action plan for weekly performance reviews
