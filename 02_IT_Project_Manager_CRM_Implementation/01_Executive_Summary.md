# Executive Summary

## Project
IT Project Manager: CRM Implementation for Sales Pipeline Visibility

## Executive Overview
The business was facing a measurable performance gap: Sales managers lacked real-time pipeline visibility because lead, opportunity, and renewal data lived in spreadsheets across three teams. The project was launched to create structure, accountability, and measurable improvement across the operating process.

## Objective
Implement a CRM workflow, migrate 4,800 active records, standardize pipeline stages, and reach 90% user adoption within 10 weeks.

## Strategy
Delivered phased CRM rollout with discovery workshops, data mapping, user acceptance testing, training, and post-launch adoption reporting.

## Outcome
The project delivered measurable improvement across the core KPIs. The strongest evidence of success was the final KPI dashboard, which shows improved performance versus baseline and target.

## Recommendation
Move CRM into standard operating rhythm and hold sales managers accountable for weekly pipeline hygiene.

## Why This Project Matters for Manager Roles
This project demonstrates the ability to diagnose a business problem, organize people around a plan, measure progress, manage risks, and communicate results in a way leaders can act on.
